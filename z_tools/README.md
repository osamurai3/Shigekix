# z_tools
Mac OSX用のz_toolsバイナリです．

## 依存しているパッケージ・ソフトウエア
- qemu
 - `brew install qemu`で入れることができます。

- bochs
 - 導入手順は[こちら](https://github.com/HariboteOS/z_tools_osx/wiki/bochs_compile)
 - Homebrewで導入できるbochsは正しく動作しないため、ソースコードからコンパイルしてインストールする必要があります。
